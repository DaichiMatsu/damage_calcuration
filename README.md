# damage_calcuration

ポケモンダメージ計算アプリ

1st_commit

・A/C、A/C努力値、B/D、B/D努力値、HP,HP努力値、わざの威力、タイプ一致、タイプ相性入力機能

・エクセルへの保存機能

・入力情報リセット機能

・計算機能（タイプを考慮しない単純なダメージ量のみ計算）

2nd_commit(計算機能大幅アップデート)

・計算機能大幅アップデート

・補正を考慮しない実数値のみのダメ計

・急所、乱数、タイプ一致、タイプ相性の計算

・五捨五超入関数作成

3rd_commit(計算機能アップデート)

・画面サイズ縦横+400

・残りHP計算&表示機能

・A/C,B/D性格補正・ランク補正入力欄追加

・A/C,B/D性格補正・ランク補正計算機能
